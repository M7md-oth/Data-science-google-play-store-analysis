# Google Play Store Data Analysis

This project analyzes the **Google Play Store dataset** to discover trends and insights about mobile applications.

The analysis is performed using Python and Jupyter Notebook.

## Dataset

The dataset contains information about Google Play Store applications such as:

* App name
* Category
* Rating
* Reviews
* Installs
* Price
* App size
* Content rating

## Analysis Performed

* Data cleaning and preprocessing
* Handling missing values
* Data visualization
* Category distribution analysis
* App ratings analysis
* Installs vs ratings insights

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook
